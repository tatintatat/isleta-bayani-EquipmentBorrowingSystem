# isleta-bayani-EquipmentBorrowedSystem

ITEL 203 – WEB SYSTEMS AND TECHNOLOGIES
PHP & MySQL CRUD Web Application

Project Title: Equipment Borrowed System                                                                                                                               
Members: Isleta, Justine Erick S.                                                                                                                                               Bayani, Hyuan Andrei U.
         
Features: A web-based platform built with PHP and MySQL that allows institutions to efficiently manage, 
track, and monitor equipment borrowing activities in real time. It consist of feature called CRUD which is Create, Read, Update, Delete
of data that is inside in our website.

Deployment Link: https://isleta-bayani-equipmentborrowingsystem.ct.ws/
